# projektarbete

## Verksamhetsbeskrivning

På Luxra erbjuder vi noggrant utvalda lyxbilar för de som låter känsla och prestige styra valet - inte budgeten. Hos oss handlar du skräddarsydda bilar med samma smidighet och trygghet som på ledande e-shoppar, men i ett helt annat segment. Luxra förenar bilvärldens exklusivitet med e-handelns enkelhet.

## Grafisk profil / UI / UX

#### Färgpalett

Vår visuella identitet bygger på en begränsad och genomtänkt färgskala enligt principen _less is more_, vilket förstärker känslan av exklusivitet.
##### Svart & vitt  
Används som basfärger för att skapa ett uttryck av elegans och tidlös exklusivitet.
##### Röd (accentfärg)  
Används sparsamt för att tillföra energi, spänning och passion. Den röda färgen fungerar som ett emotionellt förstärkningsmedel i utvalda detaljer.

#### Typografi

För att ge ett modernt, premiumorienterat och lättläst uttryck använder vi ett Typsnitt utan seriffer. Vi ger rubriker pondus och premiumkänsla genom att diskret öka mellanrummet mellan bokstäverna. 

#### Layout

Webbplatsens design ska upplevas som luftig och avskalad, där varje element har ett tydligt syfte:

- Generösa ytor och luft mellan sektioner
- En design utan distraherande dekorationer som andas exklusivitet utan att kännas överarbetad
- Begränsad mängd text; fokus på det väsentliga
